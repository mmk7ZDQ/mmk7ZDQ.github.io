教程（Windows 用户说明）

这个教程面向 Windows 用户（用 Linux 的人通常不需要这些步骤）：

  1. 先解压这个压缩包（手机相关的会另外说明）。
  2. 解压后你会看到目录，双击进入 blog 文件夹，然后打开里面的 html 文件夹。
  3. 滑到底部双击 index.html，用浏览器打开（Edge、Chrome、Firefox 任意一个都可以）。
  4. 现在你已经可以离线阅读博客了 —— 到此为止，完成！

提示：如果没有解压软件，可以下载 WinRAR 或 7-Zip。建议使用官方渠道（搜索引擎自行选择），避免下载安装携带捆绑软件的程序。

手机端说明

手机端操作也很简单（以 Android 为主，华为未单独测试）：

  1. 在手机上打开文件管理，进入 Download（下载）文件夹，找到压缩包并解压到能看到的位置。
  2. 如果系统不支持 .7z 格式，请在应用市场下载解压工具。
  3. 解压完成后，打开 blog/html 文件夹，滑到底部，用浏览器打开 index.html（建议不要使用系统自带的“HTML 查看器”）。
  4. 若使用 Chrome 或 Firefox 手机版，通常可以正常打开（不过有时排版会有问题）。
